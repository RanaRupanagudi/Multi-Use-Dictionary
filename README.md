Multi-Use Dictionary is a lightweight and user-friendly web application that allows users to explore words in multiple ways. It provides quick dictionary-style features such as searching words, viewing random entries, saving favorites, checking search history, and browsing word categories. Designed using HTML, CSS, and JavaScript, this project demonstrates foundational front-end development skills with clean navigation and simple UI components.


**Features:**

Word Search: Look up any word instantly.
Random Word Generator: Displays a random word to expand vocabulary.
Favorites: Save interesting or commonly used words.
Search History: Automatically tracks all previous searches.
Categories: Browse words grouped under different themes.
Simple UI: Minimalistic interface designed for easy navigation.

**Technology Stack:**

HTML – Page structure
CSS – Styling and layout
JavaScript – Logic and interactivity

**How to Use**

Open index.html in your browser.
Use the navigation menu to explore different features.
Search words, view random suggestions, access history, and manage favorites.
